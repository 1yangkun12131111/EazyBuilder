## 用户指南

**权限认证**

EazyBuilder支持本地认证与Ldap认证，初始密码admin/admin


LDAP认证需要手工开启，配置项（jenkins-cleint.yml）如下：
```yaml
ldap:
enable: true
url: "ldap://xxxxx:389"
base: "DC=xxx,DC=com"
userDn: "xxxxi@xxxx.com"
userPwd: "xxxx"
referral: ignore
domainName: "%s@xxxx.com"
```
